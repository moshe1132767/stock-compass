# 🧭 מצפן המניות — Stock Compass

אפליקציית ווב אישית שמחשבת המלצות קנייה/מכירה למניות אמריקאיות (NYSE/NASDAQ)
לפי 8 אינדיקטורים טכניים מוכחים, ומאחדת אותם לציון אחד (0–100) והמלצה.

## איך משתמשים
1. פותחים את הקישור בטלפון.
2. בהגדרות ⚙ מדביקים מפתח חינמי מ-[Twelve Data](https://twelvedata.com/pricing) (תוכנית Basic, חינם).
3. מוסיפים מניות לרשימה ומקבלים ציון והמלצה לכל אחת.

אפשר להוסיף למסך הבית כדי שזה יתנהג כמו אפליקציה.

## האינדיקטורים
ממוצע נע 200, צלב זהב (50/200), מומנטום 12 חודשים, MACD, RSI,
סטוכסטי, רצועות בולינגר, ונפח/זרימת כסף (OBV).

> ניתוח טכני ממוחשב למטרות מידע והשכלה בלבד — לא ייעוץ השקעות.
